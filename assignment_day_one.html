HTML Form Input Attributes

A list of all the attributes that can be used with HTML forms

  1. Action: Specifies the URL to which the form data will be submitted.

  2. Method: Specifies the HTTP method to be used when submitting the form, either "GET" or "POST".

  3. Name: Specifies a name for the form, which can be used to refer to it in scripts.

  4. Enctype: Specifies the encoding type to be used when submitting the form data. This is usually used for file upload

  5. Autocomplete: Specifies whether the browser should remember the values entered in the form field

  6. Target: Specifies the name of the window or frame in which to display the response received after submitting the form

  7. Accept-charset: Is used to specify the character encoding that the browser should use when submitting the form data to the server.

  8. Novalidate: Is used to disable the default form validation that is performed by the browser when a form is submitted.

Additionally, there are attributes that can be used with individual form elements, such as:

  1. Disabled: This attribute is used to disable the form element so that it cannot be used or edited by the user.

  2. Max: This attribute is used to specify the maximum value that can be entered in a form field. It is commonly used with number or date fields

  3. Placeholder: This attribute is used to provide a hint or example text for the user to enter into a form field.

  4. Pattern: This attribute is used to specify a regular expression pattern that the input value must match in order to be considered valid.

  5. Maxlength: This attribute is used to specify the maximum number of characters that can be entered in a text input field.

  6. Min: This attribute is used to specify the minimum value that can be entered in a form field. It is commonly used with number or date fields.

  7. Readonly: This attribute is used to make a form field read-only, meaning that the user can view the value but cannot edit it.

  8. Required: This attribute is used to make a form field required, meaning that the user must enter a value before the form can be submitted.

  9. Multiple: This attribute is used to allow the user to select multiple values from a list or dropdown menu.

  10. Size: This attribute is used to specify the size of a form field, such as the number of visible characters in a text input field or the number of options displayed in a dropdown menu.

  11. Value: This attribute is used to set the default value of a form field

  12. Step: This attribute is used to specify the increment or decrement value for number fields

  13. Minlength: This attribute is used to specify the minimum number of characters that must be entered in a text input field.
